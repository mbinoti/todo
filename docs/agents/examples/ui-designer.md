# UIDesigner — Complementos

## Padrões Essenciais
- Estado visual: `ValueListenableBuilder` ou alternativa leve para protótipos.
- Layout responsivo com `LayoutBuilder` e breakpoints declarados.
- Textos provenientes de `l10n`; respeitar tokens de tipografia/cores/spacing.
- Componentes nativos (Material 3 ou Cupertino) conforme plataforma alvo.

## Entrega sugerida
- Contexto → Achados/Oportunidades → Recomendações → Prompts/Assets → Checklist (a11y, responsividade).

- Material Design, Human Interface Guidelines, Flutter Theming.
