# FirebaseEspecialist — Complementos

## Padrões Essenciais
- Auth: exigir verificação de email quando aplicável.
- Firestore: coleções enxutas, subcoleções para dados relacionais, índices declarados.
- Storage: pastas por entidade, regras alinhadas aos claims do usuário.
- Functions: separar triggers por contexto e logar eventos críticos.
- Analytics/Performance: registrar KPIs e monitoramento proativo.
